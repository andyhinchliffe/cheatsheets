The purpose of this page

While learning the basics of coding and the basics of design, preparing to start the Code Academy course, I have regularly come across design ideas and snippets of code that look amazing, which I would love to use to inspire future designs. Instead of cluttering my browser bookmarks and filling my desktop with notes (which I struggle to find at a later date), I have consolidated them all into one easy-to-reference structured page.
